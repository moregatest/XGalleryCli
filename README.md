**XGallery Cli**
Based on Joomla! to fetch images from third party
 - Flickr
 
**How to use**
 - git clone https://github.com/jooservices/XGalleryCli.git
	 - master: stable
	 - develop: unstable
- Install database via sql file
- Execute `composer install`
- Update Flickr OAuth
- Execute /cli/xgallery.php
